# Test Generator CLI

CLI инструмент для автоматической генерации юнит-тестов с помощью GPT-4 и их запуска.

## Как работает

1. Читает JS файл из директории `source/`
2. Отправляет код в GPT-4 для анализа
3. Генерирует полный набор юнит-тестов
4. Сохраняет тесты в директорию `test/`
5. Запускает тесты через Jest
6. Показывает результаты

## Установка

```bash
npm install
```

## Настройка

Создай файл `.env` в корне проекта:

```
OPENAI_API_KEY=your_openai_api_key_here
```

## Использование

```bash
node cli.js <путь_к_файлу>
```

### Пример

```bash
node cli.js ./source/calculator.js
```

## Структура проекта

```
.
├── cli.js              # Основной CLI скрипт
├── source/             # Исходные файлы для тестирования (gitignored)
│   └── .gitkeep
├── test/               # Сгенерированные тесты (gitignored)
│   └── .gitkeep
├── utils/
│   └── logger.js       # Логирование
└── .env                # API ключи
```

## Технологии

- **OpenAI GPT-4** - генерация тестов
- **Jest** - запуск тестов
- **Node.js** - runtime

## Примечания

- Файлы в `source/` и `test/` игнорируются git'ом (кроме .gitkeep)
- Тесты генерируются с использованием лучших практик Jest
- Покрываются edge cases и error handling
